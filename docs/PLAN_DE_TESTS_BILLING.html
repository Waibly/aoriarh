1. Préparation avant de commencer

Comptes de test à créer

Utilise des alias email différents pour ne pas mélanger, par exemple ton adresse Gmail avec +testN (Gmail ignore ce qui suit le +, tous les emails arrivent dans la même boîte).

  • Compte A — à utiliser pour l'essai et la souscription Solo : ton.adresse+aoriatest1@gmail.com
  • Compte B — pour Équipe et tester les limites utilisateurs : ton.adresse+aoriatest2@gmail.com
  • Compte C — pour les invitations (sera invité par B) : ton.adresse+aoriatest3@gmail.com

Ouvre une fenêtre de navigation privée pour chaque test d'inscription, sinon tu resteras connectée avec le compte précédent.

Cartes de test Stripe à utiliser

  • Carte qui réussit : 4242 4242 4242 4242
  • Carte avec validation 3D Secure : 4000 0025 0000 3155
  • Carte qui échoue (fonds insuffisants) : 4000 0000 0000 9995
  • Carte qui échoue au renouvellement : 4000 0000 0000 0341

Pour toutes les cartes : date d'expiration au choix dans le futur (par exemple 12/29), cryptogramme à 3 chiffres au choix (par exemple 123), code postal au choix (par exemple 75001).

Après chaque test, coche "Passé" ou "Échoué" à droite. Si échoué, décris le problème dans la zone notes. Le document peut être rempli directement dans le navigateur ou imprimé en PDF.

4. Souscription à un plan payant

T06 Souscrire Solo mensuel avec succès
Prérequis
  • Être connecté avec le compte A (en plan Essai).
Étapes
  1. Aller sur /billing.
  2. Vérifier que le toggle est en "Mensuel".
  3. Cliquer sur "Souscrire Solo".
  4. Sur la page Stripe, renseigner l'adresse de facturation demandée.
  5. Vérifier qu'une ligne "TVA (20 %)" apparaît dans le récapitulatif et que le total est de 94,80 €.
  6. Renseigner la carte de test 4242 4242 4242 4242, date future, cryptogramme 123.

Résultat attendu

  • Le plan affiché est passé de "Essai" à "Solo".
  • La bannière d'essai a disparu de la sidebar.
  • Le quota mensuel est passé à 300 / 300.
  • Réception d'un email "Bienvenue sur AORIA RH Solo" avec lien vers le reçu Stripe.

  • Dans le dashboard Stripe (mode test), annuler toutes les subscriptions de test actives pour éviter les renouvellements dans les logs.
